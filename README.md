### Prerequisites
- Node.js 18+
- Backend running or use this live url https://tripmate-39hm.onrender.com/docs

### Setup
```bash
# Clone repo
git clone https://github.com/Tejashgawas/Tripmate-Frontend.git
cd tripmate-frontend
```

# Install dependencies
```
pnpm install
pnpm build
```
# Run development server
```
pnpm run dev
```
